Pristine Resume Matcher (AI-Based) - Modern Dashboard UI

A modern, dashboard-style resume screening tool designed for Pristine Technologies & Industries, Jaipur. It helps HR teams quickly match candidate resumes against a Job Description using keyword-based scoring— fully offline and client-side.

Key Highlights ✅ Modern dashboard UI ✅ No installation ✅ No backend/server ✅ Works on GitHub Pages ✅ Resume upload supported (.pdf / .docx / .txt) ✅ AI key optional
